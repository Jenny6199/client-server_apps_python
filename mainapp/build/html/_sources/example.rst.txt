Техническое описание программы
==============================

В данной документации можно ознакомиться с назначением
модулей программы и их назначением.

client_app
------------
Содержит файлы конфигурации клиентского приложения

common
------
Содержит переменные и общие функции для клиентской и серверной
части приложения

decorators
----------
Содержит код декоратора для логирования приложения.



Списки могут быть маркированными:

 * Элемент Foo
 * Элемент Bar

Или же автоматически пронумерованными:

 #. Элемент 1
 #. Элемент 2


Слова можно выделять *наклонным* или **полужирным** шрифтами.
Фрагменты кода (например, примеры команд) можно заключать в обратные кавычки, например:
команда ``sudo`` дает вам привилегии суперпользователя!
